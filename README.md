## 项目介绍
基于springboot的汇丰书店管理系统：前端 html、jquery、bootstrap，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户；集成商品列表、购物车、订单管理、商品管理等功能于一体的系统。
